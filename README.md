# Vrinda Store Sales Analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Overview
Analyzed Vrinda Store sales data to identify sales trends, customer demographics, top-performing states, and sales channels.

## Project Objectives
- Analyze monthly sales performance.
- Identify top-performing states.
- Analyze customer demographics.
- Evaluate sales channels.
- Understand customer purchasing behavior.

## Analysis Performed
- Data Cleaning
- Monthly Sales Analysis
- Gender-wise Sales Analysis
- Top States Analysis
- Channel-wise Sales Analysis
- Age Group Analysis

## Visualizations
- Monthly Sales Chart
- Gender-wise Sales Pie Chart
- Top 10 States Bar Chart
- Channel-wise Sales Chart
- Age Group Sales Chart

## Key Insights
- Women customers contributed higher sales.
- Amazon and Myntra were major sales channels.
- Customers aged 21–40 were the primary buyers.
- Top-performing states generated most of the revenue.

## Conclusion
This project demonstrates data cleaning, exploratory data analysis (EDA), and data visualization skills using Python.
