# Vrinda Store Sales Analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Overview
Analyzed Vrinda Store sales data to identify sales trends, customer demographics, top states, and sales channels.

## Key Insights
- Women contributed higher sales.
- Top states generated maximum revenue.
- Amazon and Myntra were major sales channels.
- Customers aged 21–40 were the primary buyers.
